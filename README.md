# Codewars

Repo for Codewars

Codewars에서 푼 문제를 정리한다.\
kyu는 1 ~ 8 단계로 이루어져있으며 1단계가 가장 어렵다.\
코드는 <내 풀이 - Best Practice - Clever 순서>로 정리해둔다.

### 폴더별 의미

`Fundamantals` : JS 코딩 센스와 스킬을 기르기 위한 문제
